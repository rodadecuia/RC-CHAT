---
title: "Retrieve a Contact"
openapi: "GET /contacts/{contactId}"
---

This endpoint retrieves the details of a single contact by their unique ID.

## Path Parameters

<ParamField path="contactId" type="integer" required>
  The unique identifier of the contact you want to retrieve.
</ParamField>

## Response

The response will be a single contact object.

<ResponseField name="id" type="integer">
  The unique identifier for the contact.
</ResponseField>
<ResponseField name="name" type="string">
  The name of the contact.
</ResponseField>
<ResponseField name="number" type="string">
  The contact's phone number.
</ResponseField>
<ResponseField name="email" type="string">
  The contact's email address.
</ResponseField>
<ResponseField name="isGroup" type="boolean">
  Indicates if the contact is a group.
</ResponseField>
<ResponseField name="createdAt" type="string (date-time)">
  The timestamp when the contact was created.
</ResponseField>
<ResponseField name="updatedAt" type="string (date-time)">
  The timestamp when the contact was last updated.
</ResponseField>

<RequestExample>
```bash
curl -X GET 'https://your-domain.com/api/contacts/1' \
  -H 'Authorization: Bearer YOUR_API_TOKEN'
```
</RequestExample>

<ResponseExample>
```json
{
  "id": 1,
  "name": "John Doe",
  "number": "1234567890",
  "email": "john.doe@example.com",
  "isGroup": false,
  "createdAt": "2023-10-27T10:00:00.000Z",
  "updatedAt": "2023-10-27T10:00:00.000Z"
}
```
</ResponseExample>
